ezReflections is an open source library for manipulating java classes, compiling classes and methods and searching classes structures at runtime

Features:

1. Creating class from String
2. Creating methods from String
3. Creating static methods from String
4. Searching class methods, variables and other information
5. Instantiating classes based on name or class path
6. Getting functions of classes based on name and variable types recursively
7. And many other useful tools
